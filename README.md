# Portfolio Website - Lunae

Personal portfolio website showcasing projects, skills, and experience as a Full Stack Developer.

## About

A modern, responsive portfolio built with Next.js 16 featuring smooth animations, dark/light theme toggle, and contact form integration.

## Sections

- **Hero** - Introduction with name, role, and call-to-action buttons
- **About** - Personal background and key statistics
- **Skills** - Technical skills organized by category (Frontend, Backend, Tools)
- **Projects** - Showcase of personal projects with links
- **Experience** - Work and internship history
- **Contact** - Contact form with validation

## Tech Stack

| Category | Technologies |
|----------|---------------|
| Framework | Next.js 16.2.4 |
| Language | TypeScript |
| Styling | Tailwind CSS 4 |
| Animation | Framer Motion |
| Database | MongoDB (Mongoose) |
| Forms | React Hook Form + Zod |
| Icons | Lucide React + Iconify |

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start

# Lint code
npm run lint
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
